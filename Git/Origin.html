Git远端的相关操作

查看远端仓库的相关信息

git remote -v
git remote show origin    ## 查看当前如果和远端进行推送、拉取等操作的缩写命令的含义

将本地分支推送至远端分支

git push origin  master:分支名

将远端分支同步到本地

git pull origin master(或者其他分支)

一般同步数据都是同步master的文件内容

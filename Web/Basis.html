常用模块

watch

监听某个数据是否发生变化，如果变动执行指定函数

    @Watch('Id')
    changeId(newVal: string, oldVal: string) {
        //需要操作的内容
    }

等待并发执行

使用async、await，控制函数，等待指定任务完成后在执行，一般是将异步任务顺序执行

async A(){
  await this.B()
  this.C()
}

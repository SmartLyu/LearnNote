业内标签页

在有内容需要修改的时候，需要在一个网页内弹出一个可操作展示的独立页面，详细内容

<el-dialog
        :title="update.title"
        :visible.sync="update.visible"
        width="70%" :before-close="finishUpdatee">
  // 具体内容
</el-dialog>

        update: any = {
        title: "修改",
        visible: false,
        data: {}
    }

    startUpdate(data: any) {
        this.update.visible = true
        this.update.data = data
    }

    finishUpdate() {
        this.updateRule.visible = false
        this.updateRule.data = {}
    }
  • 默认visible为false，页面会被隐藏
  • 通过调用startUpdate，将visible为true，页面即会出现
  • 调用finishUpdate，关闭页面，并将数据格式化

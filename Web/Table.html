表格

详细请查看官方文档，下面只列举几个特殊使用场景

选择某行进行操作

    <el-table
          :data="serverList"
          v-loading="loading.serverList"
          border
          ref="serverList"
          style="width: 100%;height: 100%;overflow: hidden"
          @current-change="handleCurrentChange"
          highlight-current-row>
        <el-table-column>
        </el-table-column>
  </el-table>

      handleCurrentChange(val: any) {
        // 相关操作，val即为选中行的数据对象
    }
  • v-loading 为true的时候，表格中会有个圈圈在转，表示加载中
  • highlight-current-row 高亮选中的行
  • border 带有纵向边框
  • current-change 当选择的行变化的时候触发的函数

排序

按照一定规则要求，进行排序

       <el-table-column
            sortable
            :sort-method="sortByName"
            :label="`名字`">

    sortString(value1: any, value2: any) {
        return value1.localeCompare(value2)
    }

    sortByName(row1: any, row2: any) {
        return this.sortString(row1.Name, row2.Name)
    }
  • Vue中主要是table-column内的参数
    • sortable，表格中关于该列会有排序的标签按钮
    • sort-method，指定排序的函数
  • Ts中，需要制作函数，返回对比后是升序还是降序

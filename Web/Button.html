按钮

详细请查看官方文档，下面只列举几个特殊使用场景

常用参数

<el-button type="primary" plain @click="Update(data)"
     size="small" :disabled="buttonStatus">
     {{ data ? '点击 ' + data + ' 内容' : '请先选择' }}
</el-button>
  • disabled 是否可以点击，为true的时候不可以呗点击
  • click 当点击的时候触发的函数

按钮二次确认弹框

主要使用方法popconfirm，详细链接位置

<el-popconfirm
               icon="el-icon-info"
               icon-color="red"
               title="确定修改吗？"
               :span="12"
               @confirm="postRule(item)">
  <el-button type="primary" plain style="width: 80px"
             size="small" slot="reference">
    {{ '修改' }}
  </el-button>
</el-popconfirm>
  • confirm 点击确认后的函数
  • slot 必须为reference

基本操作

判断

{% if a == b %}
   {{ a }}
{% endif %}

循环

  • 循环数组
{% for l in list %}
    {{ l }}
{% endfor %}
  • 循环字典
{% for k, v in dictory %}
    {{ k }}: {{ v }}
{% endfor %}

  • 多层循环

    主要使用的是 loop 当前循环的相关信息

字符 内容
loop.index 当前循环迭代的次数（从 1 开始）
loop.index0 当前循环迭代的次数（从 0 开始）
loop.revindex 到循环结束需要迭代的次数（从 1 开始）
loop.revindex0 到循环结束需要迭代的次数（从 0 开始）
loop.first 如果是第一次迭代，为 True 。
loop.last 如果是最后一次迭代，为 True 。
loop.length 序列中的项目数。
loop.cycle 在一串序列间期取值的辅助函数。见下面的解释。 
{% for l1 in list1 %}
    {% set outer_loop = loop %}
    {% for l2 in list2 %}
        {% if loop.index == outer_loop.index %}
             {{ l1 }}, {{ l2 }}
        {% endif %}
    {% endfor %}
{% endfor %}

{{ a | zip(b) }}

{{ a | combine(b) }}

- {{ list | to_nice_yaml | indent(2) }}
